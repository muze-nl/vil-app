Welkom bij het Virtueel Inkomstenloket!

Met het Virtueel Inkomstenloket krijg je zeer eenvoudig een overzicht van financiële regelingen die je kan aanvragen, toegekend krijgen én ontvangen. Zo krijg je automatisch het geld op je rekening gestort waar je op basis van beschikbare regelingen recht hebt. Zonder gedoe.

Met deze demonstratie laten we je zien hoe dat kan werken voor de regeling Individuele Inkomenstoeslag van de gemeente Utrecht.
