Met het Virtueel Inkomstenloket halen we de gegevens op die de overheid van jou heeft vastgelegd en opgeslagen. Met die gegevens bepalen we je mogelijke recht op inkomensondersteuning.

Soms komt het voor dat die gegevens niet kloppen. Op deze pagina krijg je daarom de mogelijkheid om je gegevens zelf aan te passen. We gebruiken dan die aangepaste gegevens om het recht op de Individuele Inkomenstoeslag te bepalen.

Hou er rekening mee dat de gemeente om bewijs kan vragen voor de aangepaste gegevens.
